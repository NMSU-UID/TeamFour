# Pico Map
Dungeon and Dragons map interface.

## Dependencies:
Python 2.7
<br>
Opencv
<br>
pep8

## Development setup
clone repo `https://github.com/NMSU_UID/pico_map.git`
<br>
install git hooks `sh install.sh`



## Project setup
Run `pip install -r requirements.txt`
<br>
Start project `python python/helloworld.py`
